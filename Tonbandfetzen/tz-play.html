NAME

tz play - generate and play sound fragments

SYNOPSIS

tz play [infile [outfile]]

DESCRIPTION

Convert infile to outfile if necessary and play it using xdg-open. Only WAVE, AIFF, MP3, and ASCII files (with tz mel or tz guitar input) are supported. This is a simple wrapper program added for convenience. Filenames default to /dev/stdin or /dev/shm/tmp.wav. The dash (-) is a shorthand for the default value.
