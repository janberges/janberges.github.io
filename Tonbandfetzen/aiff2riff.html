NAME

aiff2riff - convert .aif(f) into .wav(e) files

SYNOPSIS

aiff2riff [[infile] outfile[/]]

DESCRIPTION

Convert infile in the Audio Interchange File Format (AIFF, developed by Apple) into outfile in the Waveform Audio File Format (RIFF/WAVE, developed by IBM and Microsoft). Both store uncompressed audio data in a chunk-based container format. The main difference is that AIFF and WAVE employ the big-endian and little-endian byte order, respectively. Filenames default to /dev/stdin or /dev/stdout. The dash (-) is a shorthand for the default value. If outfile is followed by a slash (/), application-specific metadata is omitted.
