NAME

stick - concatenate audio data

SYNOPSIS

stick [[infile ...] outfile[/]]

DESCRIPTION

Concatenate the audio data from the given infiles and store the result in outfile. All files must be in the WAVE format. Filenames default to /dev/stdin or /dev/stdout. The dash (-) is a shorthand for the default value. If outfile is followed by a slash (/), application-specific metadata is omitted.
