NAME

repeat - loop audio data

SYNOPSIS

repeat [count [infile [outfile[/]]]]

DESCRIPTION

Repeat the audio data in infile count times and store the result in outfile. Fractional counts like 0.5, 1/3, or 1:4 are permitted. Negative fractional counts put the incomplete part in the beginning. All files must be in the WAVE format. count defaults to 2. Filenames default to /dev/stdin or /dev/stdout. The dash (-) is a shorthand for the default value. If outfile is followed by a slash (/), application-specific metadata is omitted.
